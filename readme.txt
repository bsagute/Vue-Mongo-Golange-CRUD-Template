Pre Requisite to run project.

A: Installed Go with version version  go1.12.7
B: Installed Node with minimun version 12.16.1
3. Installed Vue version minimum @vue/cli 4.4.1

Follow the below process

1.Extract this folder in your workspace.
2.Install node modules in .../app/ 
3.Open .../Server/GolangServer/main/config.json 
4.Set the DB details in config file
5.Install required go dependency (packages) if required
6.Run the main.go (Server)
7.Run the app client 


Note: Server is running on port 4000
